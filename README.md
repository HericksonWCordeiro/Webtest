# Sistema de Partidas

-Sistema de login e cadastro
-Somente o administrador pode criar/confirmar/editar/apagar as partidas
-Todos os usuários podem ver a lista de partidas agendadas

### Tecnologias
- PHP 🐘
- JAVASCRIPT
- MYSQL 🎲
- HTML & CSS 📄 - BOOTSTRAP 4

### Ferramentas
- Xamp (PHP, MYSQL)
- VSCode
- Git
